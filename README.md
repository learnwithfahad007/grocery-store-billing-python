# Grocery Store Billing System (Python)
This is a console-based Python program that simulates a grocery store billing system.

## Features
- Takes item name, quantity, and price
- Calculates total bill
- Applies dynamic discounts:
  - 5% if total > $50
  - 10% if total > $100
- Prints receipt-style output with aligned columns

## How to Run
python grocery_billing.py
